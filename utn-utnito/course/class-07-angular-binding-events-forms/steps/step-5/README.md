# Step 5 - Chat composer reactivo

## Espanol

### Objetivo
Conectar textarea y boton Send para capturar y enviar un mensaje local.

### Archivos a copiar
- `chat/chat.component.ts` -> copiar sobre `c07-chat-app-start/src/app/chat/chat.component.ts`
- `chat/chat.component.html` -> copiar sobre `c07-chat-app-start/src/app/chat/chat.component.html`

### Conceptos
- Binding de texto en textarea
- Submit del composer
- Boton deshabilitado por estado
- Feedback visual de ultimo mensaje
- `console.log` del mensaje cuando el submit es valido

### Validacion
Se puede enviar texto, se limpia el input y se muestra el ultimo mensaje.
