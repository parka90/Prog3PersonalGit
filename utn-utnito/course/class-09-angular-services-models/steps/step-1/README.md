# Step 1 - Demo conceptual de servicio frontend

## Espanol

### Objetivo
Mostrar el mismo concepto de estado local de clase 8, pero con datos y operaciones movidas a `MockBackendService`.

### Proyecto del step
- `services-concept-demo`

### Flujo
1. `cd steps/step-1/services-concept-demo`
2. `npm install`
3. `npm run start`

### Conceptos
- Servicio frontend como simulacion de backend.
- Componente mantiene estado y eventos de UI.
- Servicio mantiene datos y reglas del backend simulado.
- Misma UI, mejor separacion de responsabilidades.
