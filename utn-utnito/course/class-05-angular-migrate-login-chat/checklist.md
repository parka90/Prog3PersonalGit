# Checklist - Clase 05

## Espanol

- Angular app corre en `http://localhost:5300`.
- Existen `LoginComponent` y `ChatComponent`.
- `AppModule` declara ambos componentes.
- El login fue migrado a template Angular.
- El chat shell fue migrado a template Angular.
- Estado local en `AppComponent` controla pantalla activa.
- `Login` cambia de pantalla a chat.
- `Log out` vuelve de chat a login.
- El nombre escrito en login se refleja en el chat.

---

## English

- Angular app runs on `http://localhost:5300`.
- `LoginComponent` and `ChatComponent` exist.
- `AppModule` declares both components.
- Login markup is migrated to Angular template.
- Chat shell markup is migrated to Angular template.
- Local state in `AppComponent` controls active screen.
- `Login` switches to chat screen.
- `Log out` returns from chat to login.
- Username entered on login is shown in chat.
