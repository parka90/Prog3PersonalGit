# Step 1 - Conectar JavaScript al chat

## Espanol

### Objetivo
Conectar `chat.js` a `chat.html` y validar que el navegador ejecuta el script.

### Archivos a copiar
- `chat.html`
- `chat.js`

### Conceptos
- `script` externo.
- `defer`.
- Separacion HTML/CSS/JS.

### Validacion
Abrir `chat.html` y validar:
- aparece un toast no bloqueante de carga correcta,
- luego en DevTools aparece el log en consola:
`Class 04 - JavaScript loaded correctly`
