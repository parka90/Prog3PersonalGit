# c04-html-start

## Español

Proyecto inicial para Clase 04.
Punto de partida: chat estático de Clase 03 (sin JavaScript).

### Uso
1. Abrir esta carpeta en VS Code.
2. Abrir `chat.html` en navegador con doble click.
3. Aplicar cambios copiando archivos desde `../steps`.

## English

Initial project for Class 04.
Starting point: static chat from Class 03 (no JavaScript).

### Usage
1. Open this folder in VS Code.
2. Open `chat.html` in browser by double click.
3. Apply changes by copying files from `../steps`.
