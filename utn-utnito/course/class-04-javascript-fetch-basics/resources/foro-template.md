## English

### What we covered today
- Connecting JavaScript to HTML with `script` and `defer`.
- Basic DOM and events.
- Local message rendering.
- Initial async flow with `Promise` and `async/await`.
- API demo with `fetch` and JSON.

### Most common difficulties
- Script not loading due to wrong path.
- Form reloading because `preventDefault` was missing.
- Wrong DOM selectors.
- Network/endpoint failures in `fetch`.

### Reinforced best practices
- Incremental step-by-step approach.
- Visual validation after each change.
- Error handling with `try/catch`.
