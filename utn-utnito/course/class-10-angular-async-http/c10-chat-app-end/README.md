# c10-chat-app-end

Expected final snapshot for Class 10.

Includes:
- async foundation (`provideHttpClient`, `mockApiBaseUrl`, `backend_mock`),
- response models (`ResponseObject`, `Pagination`, `CreateMessageResponse`),
- `ChatApiService` for HTTP access to mock json,
- async chat flow for conversations, messages, and send,
- UI states for loading/error/sending.

## Run

```bash
npm install
npm run start
```
