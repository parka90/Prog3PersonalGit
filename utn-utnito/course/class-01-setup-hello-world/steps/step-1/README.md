# Paso 1 - Hello UTNito UI

## Español

### Objetivo
Aplicar el primer cambio de código sobre `c01-chat-app-start` para llegar a la pantalla de `Hello UTNito`.

### Archivos a copiar
Copiar estos archivos de `steps/step-1` hacia `c01-chat-app-start`:

- `src/app/app.component.ts`
- `src/app/app.component.css`

### Validación
1. Ejecutar Play/F5 desde `c01-chat-app-start`.
2. Abrir `http://localhost:5300`.
3. Verificar que aparece el título `Hello UTNito`.
